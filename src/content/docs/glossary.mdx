---
title: Glossary & Terminology
---

This document defines common terms and roles used within the SJRS Library Management System.

## Roles

- **End User**: A standard user (Student or Professor) who borrows books.
- **Librarian**: A staff member who manages day-to-day library operations (loans, returns).
- **Administrator**: A user with elevated privileges to manage users and system settings.
- **Superuser**: The highest level of access, capable of full system configuration and role management.

## System Terms

- **D1**: Cloudflare's serverless SQLite database.
- **R2**: Cloudflare's object storage (for book covers, PDFs).
- **Worker**: Serverless function that handles API requests.
- **JWT**: JSON Web Token, used for secure authentication.
