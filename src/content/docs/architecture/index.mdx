## Architecture Overview

The SJRS LMS is built as a **modern, cloud-native application** with:

- **Frontend**: React 18 + TypeScript + Vite + Ant Design
- **Backend**: Cloudflare Workers (Direct Function APIs)  
- **Database**: Cloudflare D1 (SQLite)
- **Storage**: Cloudflare R2 for file uploads
- **Real-time**: Durable Objects for live features

### Key Architectural Principles

1. **Separation of Concerns** - Clear boundaries between frontend, backend, and database
2. **Type Safety** - 100% TypeScript coverage throughout
3. **Security First** - Multi-factor authentication, CSRF protection, audit logging
4. **Performance Optimized** - Code splitting, caching, optimized queries
5. **Scalable Design** - Stateless workers, global distribution, modular architecture
