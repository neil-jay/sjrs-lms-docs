## Issues Addressed

### 1. **totalPages Bug**  →  FIXED
**Problem**: `totalPages` was incorrectly set to `pagination.total` when calling the service  
**Location**: `src/hooks/useSystemLogs.ts#L143`

**Before**:
```typescript
const response = await systemLogService.getSystemLogs(
  {
    page: pagination.pagination.current,
    pageSize: pagination.pagination.pageSize,
    total: pagination.pagination.total,
    totalPages: pagination.pagination.total //  Wrong!
  },
  filters
);
```

**After**:
```typescript
// Fix: Calculate totalPages correctly instead of passing pagination.total
const totalPages = Math.ceil(pagination.pagination.total / pagination.pagination.pageSize);
const response = await systemLogService.getSystemLogs(
  {
    page: pagination.pagination.current,
    pageSize: pagination.pagination.pageSize,
    total: pagination.pagination.total,
    totalPages //  Correct calculation
  },
  filters
);
```

**Impact**: Pagination stats now report correct page counts for UI components (e.g., "Showing 1-50 of 250" displays properly).

### 2. **No Caching/Deduplication**  →  FIXED
**Problem**: Every tab switch and filter change triggered fresh API calls, causing unnecessary load  
**Location**: `src/hooks/useSystemLogs.ts#L101-L163`

**Before** (Custom useState/useEffect Pattern):
```typescript
const [logs, setLogs] = useState<D1SystemLog[]>([]);
const [loading, setLoading] = useState(false);

useEffect(() => {
  if (autoFetch) {
    fetchLogs(); //  No caching, triggers on every dependency change
  }
}, [fetchLogs, autoFetch]);
```

**After** (React Query with Caching):
```typescript
const queryResult = useQuery({
  queryKey: ['system-logs', pagination.pagination.current, pagination.pagination.pageSize, filters],
  queryFn: async () => {
    // ... fetch logic
  },
  enabled: !!user?.id,
  staleTime: 30000, //  Cache for 30 seconds
  placeholderData: keepPreviousData, //  Prevents UI flicker
  retry: 1
});

const logs = queryResult.data?.logs || [];
const loading = queryResult.isLoading || queryResult.isFetching;
```

**Benefits**:
- **Automatic Caching**: Results cached for 30 seconds (configurable via `staleTime` option)
- **Deduplication**: Multiple components requesting same data share single request
- **No UI Flicker**: `keepPreviousData` shows previous results while fetching new page
- **Background Refetching**: Stale data refreshed in background without blocking UI
- **Automatic Retries**: Failed requests retry once automatically

## React Query Configuration

### Hook Options:
```typescript
interface UseSystemLogsOptions {
  initialPage?: number;
  initialPageSize?: number;
  staleTime?: number; // Default: 30000ms (30 seconds)
}

// Usage examples:
useSystemLogs(); // Default: 30s cache
useSystemLogs({ staleTime: 60000 }); // 60s cache for less-critical data
useSystemLogs({ staleTime: 0 }); // Disable cache (always fresh)
```

### Query Configuration:
| Setting | Value | Purpose |
|---------|-------|---------|
| **queryKey** | `['system-logs', page, pageSize, filters]` | Auto-refetch when params change |
| **staleTime** | `30000ms` (30s) | Cache duration - balances freshness vs performance |
| **placeholderData** | `keepPreviousData` | Prevents UI flicker during pagination |
| **enabled** | `!!user?.id` | Only fetch when authenticated |
| **retry** | `1` | Retry failed requests once |
| **gcTime** | Default (`5 minutes`) | How long unused cache persists in memory |

## Migration Guide (For Other Hooks)

### If you have a similar custom hook without React Query:

**Before Pattern** (Custom State Management):
```typescript
const [data, setData] = useState([]);
const [loading, setLoading] = useState(false);

useEffect(() => {
  if (autoFetch) {
    fetchData();
  }
}, [fetchData, autoFetch]);

const fetchData = async () => {
  setLoading(true);
  try {
    const response = await service.getData();
    setData(response.data);
  } finally {
    setLoading(false);
  }
};
```

**After Pattern** (React Query):
```typescript
const queryResult = useQuery({
  queryKey: ['feature-name', page, filters],
  queryFn: async () => {
    return await service.getData({ page, filters });
  },
  staleTime: 30000,
  placeholderData: keepPreviousData
});

const data = queryResult.data || [];
const loading = queryResult.isLoading || queryResult.isFetching;
```

**Benefits**: Automatic caching, deduplication, background refetching, better loading states
