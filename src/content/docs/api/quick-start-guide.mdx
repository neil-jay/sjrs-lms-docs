Get started with the SJRS LMS API in **5 minutes** with this step-by-step guide.

## Quick Start (5 Minutes)

### **1. Get Your API Credentials** ⏱ 1 min
```bash
# You'll need these environment variables
JWT_SECRET=your-secure-jwt-secret-key
MAILJET_API_KEY=your_mailjet_api_key
MAILJET_API_SECRET=your_mailjet_api_secret
```

### **2. Test the Health Endpoint** ⏱ 1 min
```bash
# Test if the API is running
curl -X GET "https://your-api-domain.com/api/health"
```

**Expected Response:**
```json
{
  "status": "ok",
  "timestamp": "..."
}
```

### **3. Register a Test User** ⏱ 1 min
```bash
curl -X POST "https://your-api-domain.com/api/auth/register" \
  -H "Content-Type: application/json" \
  -d '{
    "email": "test@example.com",
    "password": "REPLACE_WITH_YOUR_PASSWORD",
    "first_name": "Test",
    "last_name": "User",
    "user_type": "student"
  }'
```

### **4. Login (Cookie-Based Auth)** ⏱ 1 min
```bash
curl -i -c cookies.txt -X POST "https://your-api-domain.com/api/auth/login" \
  -H "Content-Type: application/json" \
  -d '{
    "email": "test@example.com",
    "password": "REPLACE_WITH_YOUR_PASSWORD"
  }'
```

This sets an HttpOnly auth cookie (default name: `secure_authToken`) in `cookies.txt`.

### **5. Make Your First API Call** ⏱ 1 min
```bash
curl -b cookies.txt -X GET "https://your-api-domain.com/api/books"
```

** Congratulations! You're now using the SJRS LMS API!**

## Setup for Development

### **Environment Setup**

#### **Option 1: Local Development**
```bash
# Clone the repository
git clone https://github.com/your-username/sjrslms.git
cd sjrslms

# Install dependencies
npm install

# Set environment variables
cp .env.example .env
# Edit .env with your credentials

# Start development server
npm run dev
```

#### **Option 2: Remote API Only**
```bash
# Just set the API base URL
export VITE_API_BASE_URL=https://your-api-domain.com
```

### **Required Environment Variables**
```bash
# JWT Configuration (choose one)
JWT_SECRET=your-secure-jwt-secret-key

# OR for key rotation (recommended)
JWT_KEYS_JSON={"key-2025-01":"REPLACE_WITH_SECRET_1","key-2025-07":"REPLACE_WITH_SECRET_2"}
JWT_DEFAULT_KID=key-2025-01
JWT_ISS=sjrslms
JWT_AUD=https://your-api-domain.com

# Email Service
MAILJET_API_KEY=your_mailjet_api_key
MAILJET_API_SECRET=your_mailjet_api_secret
MAILJET_FROM_EMAIL=noreply@sjrslms.com

# Database (if using local D1)
DB_NAME=sjrs-lms-db
```

## Client Setup Examples

### **JavaScript/TypeScript (Frontend)**
```typescript
// api-client.ts
class SJRSLMSAPI {
  private baseUrl = 'https://your-api-domain.com';

  async login(email: string, password: string) {
    const response = await fetch(`${this.baseUrl}/api/auth/login`, {
      method: 'POST',
      headers: { 'Content-Type': 'application/json' },
      credentials: 'include',
      body: JSON.stringify({ email, password })
    });

    return response.json();
  }

  async getBooks() {
    const response = await fetch(`${this.baseUrl}/api/books`, {
      credentials: 'include',
    });

    return response.json();
  }

  async bootstrapCSRF() {
    await fetch(`${this.baseUrl}/api/auth/csrf`, { credentials: 'include' });
  }
}

// Usage
const api = new SJRSLMSAPI();
await api.login('user@example.com', 'REPLACE_WITH_YOUR_PASSWORD');
await api.bootstrapCSRF();
const books = await api.getBooks();
```

### **Python**
```python
import requests

class SJRSLMSAPI:
    def __init__(self, base_url="https://your-api-domain.com"):
        self.base_url = base_url
        self.session = requests.Session()

    def login(self, email, password):
        response = self.session.post(
            f"{self.base_url}/api/auth/login",
            json={"email": email, "password": password}
        )
        return response.json()

    def bootstrap_csrf(self):
        self.session.get(f"{self.base_url}/api/auth/csrf")
        return self.session.cookies.get("XSRF-TOKEN")

    def get_books(self):
        response = self.session.get(f"{self.base_url}/api/books")
        return response.json()

# Usage
api = SJRSLMSAPI()
api.login("user@example.com", "REPLACE_WITH_YOUR_PASSWORD")
api.bootstrap_csrf()
books = api.get_books()
```

### **cURL Scripts**
```bash
#!/bin/bash
# api-test.sh

BASE_URL="https://your-api-domain.com"
EMAIL="test@example.com"
PASSWORD="REPLACE_WITH_YOUR_PASSWORD"

echo " Logging in..."
curl -s -i -c cookies.txt -X POST "$BASE_URL/api/auth/login" \
  -H "Content-Type: application/json" \
  -d "{\"email\":\"$EMAIL\",\"password\":\"$PASSWORD\"}" > /dev/null

echo " Bootstrapping CSRF..."
curl -s -b cookies.txt -c cookies.txt -X GET "$BASE_URL/api/auth/csrf" > /dev/null

echo " Getting books..."
curl -s -b cookies.txt -X GET "$BASE_URL/api/books" | jq '.'
```

## Common Issues & Solutions

### **Issue: "Invalid credentials"**
**Solution:** Check email/password spelling, ensure user exists

### **Issue: "Authentication required"**
**Solution:** Ensure cookies are enabled and you are sending the auth cookie (`credentials: 'include'` / `-b cookies.txt`).

### **Issue: "CORS error"**
**Solution:** API supports CORS, check your client configuration

### **Issue: "Rate limit exceeded"**
**Solution:** Wait a moment and retry, implement exponential backoff

### **Issue: "Service unavailable"**
**Solution:** Check system status, wait for maintenance to complete
